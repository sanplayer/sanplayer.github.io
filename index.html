Política de Privacidade

Privacidade do Usuário

O SanPlayer respeita sua privacidade. Este aplicativo é um PWA local que funciona principalmente no seu dispositivo, sem coletar dados pessoais sem seu consentimento.

Dados Armazenados

Os dados armazenados localmente incluem:

  • Playlists criadas por você
  • Favoritos salvos
  • Preferências de interface
  • Histórico de reprodução

Armazenamento Local

Todos os dados são armazenados exclusivamente no seu navegador usando localStorage e IndexedDB. Nenhum dado é enviado para servidores externos sem sua autorização.

Integração com YouTube

Este aplicativo usa a YouTube IFrame API para reproduzir conteúdo. Leia a política de privacidade do YouTube em: youtube.com/privacy

Cookies

O SanPlayer não usa cookies rastreadores. Apenas cookies técnicos necessários para o funcionamento do PWA podem ser armazenados.

Contato

Para dúvidas sobre privacidade, entre em contato através dos canais oficiais do projeto.
